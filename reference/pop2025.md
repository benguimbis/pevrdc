# A dataset containing DRC's 2025 target immunization populations at health districts levels

A dataset containing DRC's 2025 target immunization populations at
health districts levels

## Usage

``` r
pop2025
```

## Format

A data frame with 519 observations and 6 variables:

- orgunitlevel2:

  character; Province

- orgunitlevel3:

  character; Zone de santé (health district)

- naissance_mois_admin:

  numeric; monthly number of livebirths

- survivants_mois_admin:

  numeric; monthly number of surviving children

- naissance_mois_ajust:

  numeric; monthly number of livebirths ajusted

- survivant_mois_ajust:

  numeric; monthly number of surviving children ajusted

- orgunitlevel6:

  character; antenne

## Source

Programme Élargie de Vaccination de RDC.
