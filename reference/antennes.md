# A dataset containing DRC's health antennas and health districts

A dataset containing DRC's health antennas and health districts

## Usage

``` r
antennes
```

## Format

A data frame with 519 observations and 3 variables:

- province:

  character; provinces

- antenne:

  character; health antennas

- zonesante:

  character; health zones

## Source

Programme Élargie de Vaccination de RDC.
