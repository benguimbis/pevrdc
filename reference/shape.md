# A dataset with the DRC shapefile at health zone level, necessitate the sf package to load.

sf package to load.

## Usage

``` r
shape
```

## Format

A data frame with 519 observations and 5 variables:

- pays:

  character

- province:

  character

- antenne:

  character

- zonesante:

  character

- SHAPE:

  multipolygon
